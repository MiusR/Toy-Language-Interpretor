# Toy Language Interpretor

**Toy Language Interpretor** is a lightweight interpreter and debugger for a custom toy programming language.  
Designed for learning, experimentation, and exploration of language implementation concepts, this project comes with example programs out of the box so you can start interacting with the language immediately.
